## 1. Project Overview

Build a single-page-scrollable personal brand website for **Manisha Shrestha**, a professional female Emcee based in Kathmandu, Nepal. She is a public speaker, event host, brand advertiser, restaurant and product promoter, and voice-over artist. The website must position her as a premium, credible, and compelling talent that brands and event organizers want to book.

The tone is **Elegant & Sophisticated** — refined, confident, and aspirational, while still warm and approachable. Think: editorial luxury meets personal charisma.

---

## 2. Design System

### 2.1 Theme Direction

**"Noir Gold" — Dark Luxury Editorial**

The site uses a dark, richly atmospheric base that communicates sophistication and stage presence. Gold and champagne accents reflect glamour. Generous white space and restrained typography signal professionalism. The overall feel is that of a high-end talent portfolio — like a fashion editorial crossed with a luxury brand lookbook.

**Mood keywords:** Dramatic. Confident. Polished. Luminous. Feminine power.

---

### 2.2 Color Palette

| Role | Name | Hex |
|---|---|---|
| Background (Primary) | Noir | `#0E0E0E` |
| Background (Secondary) | Deep Charcoal | `#1A1A1A` |
| Background (Card/Surface) | Warm Graphite | `#242424` |
| Primary Accent | Champagne Gold | `#C9A96E` |
| Secondary Accent | Rose Gold | `#B76E79` |
| Text (Primary) | Ivory White | `#F5F0E8` |
| Text (Secondary) | Soft Warm Gray | `#A89F96` |
| Border / Divider | Dark Gold Tint | `#3A3020` |
| CTA Button | Champagne Gold | `#C9A96E` |
| CTA Button Text | Noir | `#0E0E0E` |
| Highlight / Hover | Pale Gold | `#E8D5A3` |

**Usage rules:**
- Background is always dark (Noir or Deep Charcoal). Never use a white or light background.
- Gold is used sparingly as an accent — for headings, underlines, borders, icons, and CTAs only.
- Rose Gold is used as a softer secondary accent — for tags, subtle hover states, and decorative elements.
- All body text must pass WCAG AA contrast on dark backgrounds.

---

### 2.3 Typography

| Role | Font | Weight | Notes |
|---|---|---|---|
| Display / Hero Heading | **Cormorant Garamond** | 300–700 | Elegant serif. Used for large hero text and section titles. Import from Google Fonts. |
| Section Subheadings | **Cormorant Garamond** | 600 Italic | Used for service names and card headings. |
| Body Copy | **Jost** | 300–400 | Clean, modern sans-serif. Excellent readability on dark backgrounds. |
| Navigation | **Jost** | 400–500 | All caps, wide letter-spacing (0.15em). |
| Accent / Decorative | **Pinyon Script** | 400 | Ornate script. Used only for decorative single words or tagline highlights (e.g., "Manisha"). |
| Labels / Tags / Metadata | **Jost** | 300 | Small caps or uppercase, muted color. |

**Type scale (desktop):**
- Hero title: `clamp(3.5rem, 8vw, 7rem)`
- Section heading: `clamp(2rem, 4vw, 3.5rem)`
- Subheading: `1.5rem`
- Body: `1rem` / line-height `1.8`
- Caption / label: `0.8rem`

---

### 2.4 Visual Style Details

- **Textures:** Subtle grain/noise overlay at 3–5% opacity on backgrounds to add depth and warmth.
- **Borders:** Thin 1px gold-tinted borders (`#3A3020`) on cards and dividers.
- **Shadows:** Use deep, warm shadows (`rgba(0,0,0,0.6)`) rather than blue-tinted shadows.
- **Images:** All images use a subtle warm-gold overlay/duotone treatment to maintain palette cohesion. Portrait photos should feel editorial — high contrast, cinematic.
- **Animations:** Smooth fade-up reveals on scroll (staggered, 0.2s delay between elements). Hover states on cards use subtle upward lift (`translateY(-4px)`) with a faint gold glow. No flashy or distracting animations.
- **Decorative elements:** Thin horizontal gold lines as section separators. Optional: subtle geometric gold corner ornaments on featured cards.
- **Cursor:** Custom cursor — a small gold dot with a larger circle outline that scales on hover over interactive elements.

## 4. Page Sections — Detailed Specifications

### 4.1 Navigation Bar

**Behavior:**
- Fixed/sticky to top of viewport on scroll.
- On load: transparent with white text (overlaying hero).
- On scroll (past 80px): transitions to a semi-transparent dark background (`rgba(14,14,14,0.92)`) with a thin gold bottom border. Backdrop blur effect (`backdrop-filter: blur(12px)`).
- Mobile: collapses into a hamburger menu. Menu opens as a full-screen dark overlay with centered navigation links.

**Content:**
- Left: Logo — "Manisha Shrestha" in Cormorant Garamond, with "MS" monogram as a compact logo mark.
- Center (desktop): Navigation links — `About`, `Services`, `Portfolio`, `Contact`
- Right: A gold CTA button — **"Book Now"** — that smooth-scrolls to the Contact section.
- Social icons: Instagram and TikTok icon links (SVG), placed beside the CTA or in the hamburger menu on mobile.

## 5. Technical Requirements

### 5.1 Stack

- **Framework:** Next.js 14+ (App Router) or plain HTML/CSS/JS if simpler deployment is preferred.
- **Styling:** Tailwind CSS + custom CSS variables for the design system tokens.
- **Animations:** Framer Motion (if React/Next.js) or GSAP ScrollTrigger for scroll-based reveals.
- **Icons:** Lucide React or Heroicons SVG set.
- **Fonts:** Google Fonts — `Cormorant Garamond`, `Jost`, `Pinyon Script`.
- **Images:** Next.js `<Image />` component with lazy loading, or native `loading="lazy"` in HTML.
- **Form handling:** Client-side validation + Formspree or EmailJS for form submission (no backend required).
- **Video embeds:** Standard YouTube iframe embed with `nocookie` domain (`youtube-nocookie.com`) for privacy.

### 5.2 Performance
- All images compressed (WebP format preferred, max 300KB per image).
- Fonts loaded with `display: swap`.
- Lazy load all images and video iframes below the fold.
- Target Lighthouse score: 90+ on Performance, Accessibility, Best Practices.

### 5.3 Responsiveness
- Fully responsive: Mobile (320px+), Tablet (768px+), Desktop (1280px+).
- Mobile-first CSS approach.
- Touch-friendly tap targets (min 44x44px).
- Hamburger menu on mobile/tablet.

### 5.4 SEO
- Semantic HTML5 elements (`<header>`, `<main>`, `<section>`, `<footer>`, `<nav>`).
- Meta title: `"Manisha Shrestha | Emcee, Voice Artist & Brand Ambassador — Kathmandu, Nepal"`
- Meta description: `"Manisha Shrestha is a professional Emcee, Voice Over Artist and Brand Ambassador based in Kathmandu, Nepal. Available for events, campaigns, and speaking engagements."`
- OG/Twitter card meta tags for social sharing.
- Structured data (JSON-LD): `Person` schema with name, jobTitle, url, sameAs (social profiles).

### 5.5 Accessibility
- All images have descriptive `alt` text.
- Color contrast WCAG AA compliant for all text.
- Keyboard navigable (focus-visible outlines).
- ARIA labels on icon-only buttons.
- Form fields have associated `<label>` elements.

## 9. Interaction & Animation Specs

| Element | Animation |
|---|---|
| Section entry | Fade up + opacity 0→1, triggered on scroll into view (IntersectionObserver or Framer Motion `whileInView`) |
| Stagger delay | 0.15s between sibling elements (cards, stats, nav links) |
| CTA button hover | Background lightens to `--color-gold-pale`, subtle scale `1.02` |
| Portfolio card hover | `translateY(-6px)`, faint gold box-shadow |
| Nav link hover | Gold underline slides in from left |
| Form field focus | Bottom-border transitions from muted to gold |
| Lightbox open/close | Fade in/out with slight scale (`0.95 → 1`) |
| Hamburger menu | Full-screen overlay slides down from top |
| Custom cursor | Gold dot tracks pointer; expands on hover over links/buttons |
